Welcome to Artworks API!

Django Rest Framework should make using the API simple, but for manual testing here are the endpoints available:

Available Endpoints

GET: /artworks/

Returns all artworks in the database

GET: /artists

Returns all artists in the database

GET: /artworks/:id

Returns the details of the artwork with the specified id

GET: /artists/:id

Returns the details of the artist with the specified id

POST: /artists/

Use this to add an artist. The request body should be in JSON and contain all of the following properties:

  • "name" (a string)
  • "birth_year" (an integer)
  • "birth_place" (a string)

POST: /artworks/

Use this to add an artwork. The request body should be in JSON and contain all of the following properties:

  • "title" (a string)
  • "year" (an integer)
  • "artist_id" (an integer)

The given artist_id must exist in the artists database.

PATCH/PUT: /artworks/:artworkId

Used for updating artwork information. Request body has the same format as the POST method, but not all parameters are required for PATCH

PATCH/PUT: /artists/:artistId

Used for updating artist information. Request body has the same format as the POST method, but not all parameters are required for PATCH

DELETE: /artworks/:artworkId

Use this to delete an artwork.

DELETE: /artists/:artistId

Use this to delete an artist.
